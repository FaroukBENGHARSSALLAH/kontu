Kontu
==========================

Spring Boot web module to manage client bank accounts.

Used APIs,

- Java 8
- Spring Boot 1.5.1
- Spring Data 1.11
- JPA, Hibernate 5.0
- H2database
- Spring Security 4.2
- HTML5, CSS3, Boostrap 3, JQuery 2

You can visit the live version via [Kontu](https://kontu-dev-alpha.herokuapp.com).

> This project is in the alpha stage
